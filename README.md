## Objective
To simulate real-time procurement business process using SAP MM.

## Business Scenario
A manufacturing company procures raw materials from registered vendors.

## Modules Used
- SAP MM
- SAP FI

## Process Flow

| Step | Transaction Code | Description |
|------|------------------|-------------|
| 1 | XK01 | Vendor Master Creation |
| 2 | MM01 | Material Master Creation |
| 3 | ME11 | Purchasing Info Record |
| 4 | ME21N | Purchase Order Creation |
| 5 | MIGO | Goods Receipt Posting |
| 6 | MIRO | Invoice Verification |
| 7 | FB03 | Accounting Document Verification |

## Result
Successfully completed full Procure to Pay cycle and verified accounting integration.
